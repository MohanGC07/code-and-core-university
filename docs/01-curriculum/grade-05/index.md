## Grade 5 Curriculum Overview

## Theme

### Discover Technology Through Creativity

Grade 5 introduces learners to the exciting world of technology. Students develop confidence using computers, learn how to think logically, explore programming through visual tools, discover the basics of artificial intelligence and robotics, and complete hands-on projects that encourage curiosity and creativity.

---

## Grade Profile

| Attribute | Description |
| ----------- | ------------- |
| Grade | 5 |
| Academic Stage | Foundation |
| Typical Age | 10–11 years |
| Learning Style | Hands-on, visual, interactive |
| Programming Level | Beginner |
| Project Level | Introductory |

---

## Annual Goal

By the end of Grade 5, learners should be able to:

- Use a computer confidently.
- Practice safe and responsible digital citizenship.
- Apply computational thinking to simple problems.
- Create interactive programs using Scratch and PictoBlox.
- Explain basic AI concepts using everyday examples.
- Build simple robotics activities.
- Work effectively with classmates.
- Present their projects with confidence.

---

## Learning Outcomes

Students will be able to:

## Digital Literacy

- Identify computer hardware.
- Use keyboard and mouse efficiently.
- Create, save, and organize files.
- Browse the internet safely.
- Understand basic cybersecurity habits.

---

## Computational Thinking

- Recognize patterns.
- Break problems into smaller parts.
- Write simple algorithms.
- Create flowcharts.
- Debug simple logical mistakes.

## Assessment Strategy

Assessment is based primarily on practical work.

|Assessment|Weight|
|------------|-------:|
|Classroom Activities|20%|
|Programming Exercises|20%|
|Robotics Activities|15%|
|Mini Projects|20%|
|Semester Project|15%|
|Portfolio & Presentation|10%|

---

## Portfolio Requirements

Every learner maintains a portfolio including:

- Minimum 5 Scratch projects
- Minimum 2 robotics projects
- Reflection journal
- Presentation slides or posters
- Photos of completed work
- Certificates of participation (if applicable)
