## Grade 7 Semester 1 Learning Plan

## Semester Theme

## Building Strong Software Foundations

Semester 1 develops students' confidence in computational thinking, Python programming, and modern web development. Learners begin applying engineering principles through practical coding projects and collaborative activities.

---

## Semester Overview

| Duration | 6 Months |
| ----------- | ----------- |
| Programming Language | Python |
| Web Technologies | HTML5 & CSS3 |
| Teaching Method | Project-Based Learning |
| Semester Project | Personal Portfolio Website |

---

## Learning Objectives

By the end of Semester 1, students will be able to:

- Apply computational thinking to solve problems.
- Develop structured Python programs.
- Create reusable functions.
- Store and manipulate data using collections.
- Read from and write to text files.
- Design responsive web pages using HTML and CSS.
- Document projects professionally.
- Present completed work confidently.
