## Grade 7 Yearly Learning Plan

## Purpose

This document provides the yearly instructional roadmap for Grade 7. The curriculum is divided into two semesters, ensuring a gradual progression from intermediate programming concepts to engineering-oriented problem solving.

---

## Annual Theme

## From Coding to Engineering

Students strengthen programming skills while expanding into web development, robotics, artificial intelligence, and software engineering practices.

---

## Annual Learning Goals

By the end of the academic year, learners will be able to:

- Develop structured Python applications.
- Design responsive websites.
- Apply computational thinking to solve problems.
- Build Arduino-based automation projects.
- Understand AI concepts and ethics.
- Document software professionally.
- Present technical solutions confidently.

## Teaching Methodology

Instruction follows the Code and Core learning model:

1. Learn
2. Observe
3. Practice
4. Build
5. Test
6. Improve
7. Present
8. Reflect

Students spend approximately:

| Activity        | Percentage |
|:----------------|-----------:|
| Theory          |        30% |
| Practical Work  |        70% |
