## Grade 7 Semester 2 Learning Plan

## Semester Theme

## Engineering Intelligent Systems

Semester 2 introduces learners to Artificial Intelligence, Robotics, Computer Systems, and Software Engineering concepts. Students integrate programming, electronics, and engineering principles to build practical technology solutions.

---

## Semester Overview

| Attribute | Description |
| ----------- | ------------- |
| Duration | 6 Months |
| Programming Language | Python |
| Robotics Platform | Arduino UNO |
| Learning Approach | Project-Based Learning |
| Semester Project | Smart Classroom Automation Prototype |

---

## Learning Objectives

By the end of Semester 2, students will be able to:

- Explain the fundamentals of Artificial Intelligence.
- Build Arduino-based automation projects.
- Interface sensors and actuators.
- Understand computer hardware and networking fundamentals.
- Apply engineering design principles.
- Document technical projects professionally.
- Collaborate effectively in teams.
- Present innovative solutions with confidence.

---

## Unit 5 — Artificial Intelligence Foundations

## Duration

Weeks 1–4

### Topics

- Introduction to Artificial Intelligence
- Machine Learning Concepts
- AI vs Traditional Programming
- Data and AI
- Computer Vision (Concept)
- Natural Language Processing (Concept)
- AI Ethics
- Responsible AI

### Practical Activities

- AI Application Research
- AI Ethics Discussion
- Image Classification Demonstration
- AI Reflection Journal

### Learning Evidence

Students should demonstrate the ability to:

- Explain AI concepts using real-world examples.
- Differentiate AI from traditional software.
- Discuss ethical responsibilities when using AI.

## Unit 6 — Robotics & Embedded Systems

### Duration — Robotics & Embedded Systems

Weeks 5–9

### Topics — Robotics & Embedded Systems

- Arduino Architecture
- Digital Inputs and Outputs
- Analog Inputs
- LEDs
- Push Buttons
- Buzzers
- Servo Motors

### Practical Activities — Robotics & Embedded Systems

- LED Controller
- Digital Dice
- Servo Door System
- Reaction Timer

### Learning Evidence — Robotics & Embedded Systems

Students should demonstrate the ability to:

- Build electronic circuits safely.
- Upload Arduino programs.
- Debug hardware and software issues.

## Semester Project

## Smart Classroom Automation Prototype

Students work in teams to design and develop a classroom automation solution.

### Suggested Features

- Automatic Lighting
- Attendance Indicator
- Temperature Monitoring
- Distance-Based Door Alert
- Classroom Status Display

### Deliverables

- Working Prototype
- Circuit Diagram
- Source Code
- Project Documentation
- User Guide
- Final Presentation

## Teaching Methodology

Every lesson follows the engineering learning cycle.

1. Discover
2. Explore
3. Design
4. Build
5. Test
6. Improve
7. Document
8. Present
