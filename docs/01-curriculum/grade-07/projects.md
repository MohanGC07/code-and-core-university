## Grade 7 Projects & Innovation Framework

## Purpose

Project-Based Learning (PBL) is the core instructional strategy of Grade 7. Every learner applies classroom knowledge to practical engineering challenges through progressively complex projects.

Projects encourage:

- Critical Thinking
- Creativity
- Engineering Design
- Communication
- Team Collaboration
- Problem Solving
- Technical Documentation

---

## Project Categories

The Grade 7 curriculum consists of six project categories.

| Category | Description |
| -------- | ----------- |
| Programming | Python software projects |
| Web Development | HTML & CSS websites |
| Artificial Intelligence | AI exploration activities |
| Robotics | Arduino engineering projects |
| Engineering Design | Real-world problem solving |
| Capstone | Semester-end innovation projects |

## Innovation Showcase

At the end of the academic year, every learner presents their work during the Grade 7 Innovation Showcase.

Presentation should include:

- Problem Statement
- Solution Design
- Demonstration
- Challenges
- Lessons Learned
- Future Improvements

---

## Project Documentation Requirements

Every project must include:

- Project Title
- Objectives
- Materials Used
- Design Process
- Source Code
- Circuit Diagram (if applicable)
- Testing Results
- Challenges
- Improvements
- Reflection

---

## Assessment Rubric

| Criteria | Weight |
| ---------- | -------: |
| Technical Implementation | 30% |
| Problem Solving | 20% |
| Creativity & Innovation | 15% |
| Documentation | 15% |
| Presentation | 10% |
| Teamwork | 10% |

---

## Portfolio Evidence

Each completed project should include:

- Source Code
- Screenshots
- Photos
- Videos (optional)
- Documentation
- Reflection Journal

---

## Success Criteria

A successful Grade 7 project should:

- Solve a clearly defined problem.
- Demonstrate technical understanding.
- Apply classroom learning.
- Follow engineering design principles.
- Include complete documentation.
- Be presented confidently.
- Reflect on future improvements.
