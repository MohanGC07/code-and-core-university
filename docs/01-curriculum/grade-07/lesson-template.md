## Lesson Introduction (10 Minutes)

## Teacher Activities

- Welcome students.
- Review previous lesson.
- Introduce today's objectives.
- Connect learning to real-world applications.

## Student Activities

- Participate in discussion.
- Answer review questions.
- Share prior knowledge.

---

## Concept Explanation (20 Minutes)

The teacher explains:

- Key concepts
- Technical vocabulary
- Demonstrations
- Best practices
- Common mistakes

Students should actively ask questions throughout the explanation.

---

## Guided Practice (20 Minutes)

Students complete structured activities with teacher support.

Examples include:

- Coding exercises
- HTML practice
- Circuit assembly
- AI demonstrations
- Debugging activities

Teacher circulates and provides guidance as needed.

---

## Independent Practice (30 Minutes)

Students apply today's learning independently.

Examples:

- Build a Python program
- Create a webpage
- Assemble a circuit
- Modify an existing project
- Complete a design challenge

---

## Challenge Activity (Optional)

Students who finish early may attempt an extension task.

Examples:

- Add extra features
- Improve user interface
- Optimize code
- Create additional functionality
- Explain their solution to classmates

---

## Classroom Discussion

Suggested discussion questions:

- What problem did today's lesson solve?
- What challenges did you encounter?
- Which solution worked best?
- How could this project be improved?
- Where could this technology be used?

---

## Assessment

## Formative Assessment

Teacher observes:

- Participation
- Problem-solving
- Collaboration
- Practical skills

---

## Success Criteria

Students successfully complete the lesson when they can:

- Explain today's concepts.
- Complete the practical activity.
- Demonstrate working solutions.
- Answer review questions.
- Document their work.

---

## Differentiation

## Support

Students requiring additional support may receive:

- Guided examples
- Pair programming
- Teacher assistance
- Simplified exercises

---

## Extension

Advanced learners may:

- Complete challenge activities
- Explore additional features
- Help peers
- Research related technologies

---

## Homework

Examples:

- Complete unfinished project
- Practice coding exercises
- Improve documentation
- Watch recommended learning videos
- Prepare for next lesson

## Reflection

Students complete a short reflection.

### Questions

- What did I learn today?
- What was most challenging?
- How did I solve problems?
- What do I still need to practice?
- What am I proud of today?
