## Grade 7 Learning Resources

## Purpose

This document defines the recommended learning resources for Grade 7. These resources support classroom instruction, project-based learning, independent practice, and continuous skill development.

The goal is to provide students with access to professional tools while maintaining a beginner-friendly learning environment.

---

## Resource Categories

The Grade 7 curriculum uses resources from the following categories.

| Category | Purpose |
| ---------- | --------- |
| Software | Programming and development |
| Hardware | Robotics and electronics |
| Learning Materials | Teaching and learning support |
| Documentation | Professional documentation |
| Classroom Equipment | Practical learning |
| Digital Resources | Research and collaboration |

---

## Software Resources

## Programming

Students use Python for developing applications.

### Software

- Python 3
- IDLE (Introduction)
- Visual Studio Code

### Skills

Students should learn to:

- Create projects
- Run programs
- Debug code
- Organize files

---

## Web Development

Students build websites using:

- HTML5
- CSS3

Recommended Browser

- Google Chrome
- Microsoft Edge
- Mozilla Firefox

Students learn how modern web pages are structured and styled.

---

## Robotics

Students use:

- Arduino IDE

Programming includes:

- Uploading sketches
- Reading Serial Monitor
- Basic debugging

---

## Documentation

Students maintain professional documentation using:

- Markdown
- PDF Reports
- Presentation Slides

Students should organize documentation clearly and consistently.

## Hardware Resources

## Student Computer

Recommended specifications:

| Component | Recommendation |
| --------- | --------------- |
| Processor | Intel Core i3 / AMD Ryzen 3 or higher |
| RAM | 8 GB Minimum |
| Storage | 256 GB SSD or higher |
| Display | 14" or larger |
| Internet | Stable Broadband |

## Recommended Classroom Practices

Teachers should:

- Demonstrate before assigning.
- Encourage experimentation.
- Promote collaborative learning.
- Allow students to debug independently.
- Encourage documentation after every project.

---

## Resource Management

Students should learn to:

- Organize project folders.
- Back up important work.
- Label files consistently.
- Maintain hardware responsibly.
- Return equipment after use.

---

## Laboratory Safety Resources

Every robotics laboratory should include:

- First Aid Kit
- Fire Extinguisher
- Electrical Safety Instructions
- Equipment Inventory
- Emergency Contact Information

Students should never:

- Touch exposed electrical connections.
- Force electronic components.
- Connect circuits without teacher approval.
- Leave powered circuits unattended.

---

## Portfolio Resources

Students should maintain evidence including:

- Source Code
- Project Reports

- Reflection Journals
- Presentation Slides

The curriculum team should review resources annually to:

- Replace damaged equipment.
- Update software versions.
- Improve learning materials.
- Expand robotics equipment.
