## Grade 7 Curriculum Overview

## Theme

## From Coding to Engineering

Grade 7 marks the transition from foundational computing to engineering-oriented learning. Students strengthen their programming skills, begin professional web development, explore artificial intelligence, develop robotics systems, and learn how modern software is designed and documented.

The curriculum emphasizes curiosity, experimentation, collaboration, and problem-solving through hands-on projects.

| Attribute | Description |
| --------- | ----------- |
| Grade | 7 |

| Programming Level | Intermediate |
| Learning Style | Project-Based |
| Collaboration | Individual & Team |

---

## Annual Goal

By the end of Grade 7 every learner should be able to:

- Build structured Python applications.
- Create responsive websites using HTML and CSS.
- Understand the basic principles of Artificial Intelligence.
- Build Arduino-based automation projects.
- Apply computational thinking to real-world problems.
- Document technical work professionally.
- Collaborate effectively with classmates.
- Prepare confidently for Grade 8 technologies.

---

## Learning Philosophy

Learning follows the Code and Core educational principles:

- Learn by Building
- Think Before Coding
- Practice Every Lesson
- Collaborate Respectfully
- Reflect Frequently
- Document Everything
- Build Real Projects
- Learn Responsibly

Students spend approximately:

| Learning Activity | Percentage |
| --- | ---: |
| Theory | 30% |
| Practical Work | 70% |

## Curriculum Domains

The Grade 7 curriculum consists of seven learning domains.

| Domain | Focus |
| --------- | ------- |
| Programming | Intermediate Python |
| Web Development | HTML5 & CSS3 |
| Artificial Intelligence | AI Foundations |
| Robotics | Arduino & Sensors |
| Computer Systems | Hardware & Networking |
| Software Engineering | Documentation & Version Control Concepts |
| Innovation Studio | Team-Based Projects |

---

## Programming

Students expand their Python knowledge by learning:

- Functions
- Lists
- Tuples
- Dictionaries
- File Handling
- Error Handling
- Modular Programming
- Problem Solving

Students learn to write cleaner, reusable, and maintainable code.

---

## Web Development

Students begin creating modern websites using:

- HTML5
- CSS3
- Responsive Design
- Forms
- Tables
- Images
- Multimedia
- Page Layout

Students complete multiple web-based projects throughout the year.

---

## Artificial Intelligence

Students explore:

- What is AI?
- Machine Learning Concepts
- Data and AI
- Computer Vision (Introduction)
- Natural Language Processing (Introduction)
- AI Ethics
- Responsible AI

The emphasis is on understanding concepts rather than training complex models.

---

## Robotics

Students work with Arduino-based systems including:

- LEDs
- Buttons
- Buzzers
- Servo Motors
- Ultrasonic Sensors
- LCD Displays
- Analog Sensors

Projects combine programming with electronics to solve practical problems.

---

## Computer Systems

Students develop an understanding of:

- Computer Hardware
- CPU
- RAM
- Storage
- Operating Systems
- Networking Basics
- Internet Communication

---

## Software Engineering

Students are introduced to professional development practices including:

- Project Organization
- Code Readability
- Technical Documentation
- Markdown
- Version Control Concepts
- Collaborative Development

---

## Major Projects

Semester 1

- Personal Portfolio Website
- Python Quiz Application
- Student Information Program
- Interactive Educational Website

Semester 2

- Smart Parking System
- Weather Monitoring Station
- Smart Classroom Automation
- AI Image Classification Demonstration
- Innovation Challenge Project

---

## Assessment Strategy

| Component | Weight |
| --------- | -------: |
| Classroom Activities | 15% |
| Programming Exercises | 20% |
| Web Development Projects | 15% |
| Robotics Laboratories | 15% |
| Major Projects | 20% |
| Portfolio | 10% |
| Presentation | 5% |

Assessment prioritizes practical application, creativity, teamwork, and continuous improvement.

---

## Portfolio Requirements

Every learner maintains a professional portfolio containing:

- Python Programs
- Website Projects
- Robotics Projects
- Technical Documentation
- Reflection Journals
- Presentation Materials
- Project Reports

The portfolio demonstrates growth throughout the academic year.

---

## Expected Graduate Profile

A Grade 7 learner should demonstrate:

- Logical thinking
- Programming confidence
- Creativity
- Engineering mindset
- Communication skills
- Team collaboration
- Responsible technology use
- Curiosity for continuous learning

---

## Progression to Grade 8

Completion of Grade 7 prepares learners for:

- Object-Oriented Programming
- JavaScript
- Databases
- Internet of Things (IoT)
- Intermediate Artificial Intelligence
- Larger Software Engineering Projects
