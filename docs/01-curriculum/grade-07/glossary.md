## Grade 7 Glossary

## Purpose

This glossary provides standard definitions for the technical terms introduced during Grade 7. It ensures students, teachers, and parents use a consistent vocabulary across programming, web development, artificial intelligence, robotics, computer systems, and software engineering.

---

## A

## Algorithm

A step-by-step procedure used to solve a problem or complete a task.

---

## Artificial Intelligence (AI)

The field of computer science that enables computers to perform tasks that normally require human intelligence, such as recognizing images, understanding language, or making decisions.

---

## Arduino

An open-source electronics platform consisting of programmable microcontroller boards used to build interactive electronic projects.

---

## Array

A collection of related values stored together. In Python, lists are commonly used instead of traditional arrays.

---

## B

## Boolean

A data type that has only two possible values:

- True
- False

---

## Bug

An error or mistake in software or hardware that causes incorrect behavior.

---

## Breadboard

A reusable board used for building electronic circuits without soldering.

---

## C

## Class

A blueprint used in object-oriented programming to create objects.

---

## Cloud Storage

Online storage that allows files to be saved and accessed through the internet.

---

## Code

Instructions written in a programming language that tell a computer what to do.

---

## Computational Thinking

A structured approach to solving problems using logical reasoning, decomposition, abstraction, and algorithms.

---

## CSS (Cascading Style Sheets)

A language used to control the appearance and layout of web pages.

---

## D

## Data

Facts, values, or information processed by a computer.

---

## Debugging

The process of finding and correcting errors in a program.

---

## Dictionary

A Python data structure that stores information using key-value pairs.

---

## Digital Citizenship

Responsible, safe, ethical, and respectful use of digital technologies.

---

## E

## Electronics

The study and application of electrical circuits and electronic components.

---

## Error

A mistake in code or hardware that prevents correct operation.

---

## F

## File Handling

Reading, writing, and managing files using a programming language.

---

## Flowchart

A graphical representation of an algorithm using symbols and arrows.

---

## Function

A reusable block of code designed to perform a specific task.

---

## H

## Hardware

The physical components of a computer system.

---

## HTML (HyperText Markup Language)

The standard markup language used to structure web pages.

---

## I

## IDE (Integrated Development Environment)

Software used to write, run, debug, and manage programs.

Examples include:

- Visual Studio Code
- Arduino IDE

---

## Input

Information entered into a computer program.

---

## Internet

A worldwide network connecting computers and devices.

---

## L

## LCD (Liquid Crystal Display)

A display module used to show text or numbers in electronics projects.

---

## LED (Light Emitting Diode)

A small electronic component that produces light when electrical current passes through it.

---

## List

An ordered collection of values in Python.

---

## Loop

A programming structure that repeats a block of code multiple times.

---

## M

## Machine Learning

A branch of Artificial Intelligence where computers learn patterns from data to make predictions or decisions.

---

## Microcontroller

A compact programmable computer used to control electronic devices.

Example:

Arduino UNO

---

## Module

A file containing reusable Python code that can be imported into another program.

---

## N

## Network

A group of connected computers or devices that communicate with each other.

---

## O

## Object

An instance created from a class in object-oriented programming.

---

## Operating System

Software that manages computer hardware and software resources.

Examples include:

- Windows
- Linux
- macOS

---

## P

## Portfolio

A structured collection of projects and evidence demonstrating a student's learning and achievements.

---

## Programming

The process of writing instructions for a computer to perform tasks.

---

## Python

A high-level programming language used throughout the Grade 7 curriculum.

---

## R

## RAM (Random Access Memory)

Temporary memory used by a computer while programs are running.

---

## Resistor

An electronic component that limits the flow of electrical current.

---

## Robotics

The design, construction, and programming of robots and automated systems.

---

## S

## Sensor

An electronic component that detects changes in the environment.

Examples include:

- Ultrasonic Sensor
- LDR Sensor
- Temperature Sensor

---

## Servo Motor

A motor capable of moving to precise positions using programmed instructions.

---

## Software

Programs and applications that run on a computer.

---

## Source Code

The human-readable instructions written by programmers.

---

## T

## Troubleshooting

The process of identifying and solving technical problems.

---

## Tuple

An ordered collection in Python whose values cannot be modified after creation.

---

## U

## Ultrasonic Sensor

A sensor that measures distance using ultrasonic sound waves.

---

## User Interface (UI)

The visual elements through which users interact with software.

---

## V

## Variable

A named location used to store data in a program.

---

## Version Control

A system used to track changes made to files and software projects over time.

---

## W

## Web Browser

Software used to access websites.

Examples include:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox

---

## Website

A collection of interconnected web pages accessible through the internet.
