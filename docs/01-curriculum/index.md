## Curriculum Overview

## Purpose

The Code and Core curriculum provides structured, project-based learning pathways that prepare learners for academic success and industry careers.

## Learning Philosophy

Every learning path follows the same framework:

1. Learn Concepts
2. Practice Skills
3. Build Projects
4. Complete Assessments
5. Create a Portfolio
6. Earn Certification
7. Contribute to the Community
