## Grade 8 Glossary

## Purpose

This glossary provides concise definitions of the technical terms introduced throughout the Grade 8 curriculum. Students are encouraged to understand and correctly use these terms during classroom discussions, practical activities, projects, and presentations.

---

## A

## Algorithm

A sequence of logical steps used to solve a problem or perform a task.

---

## Artificial Intelligence (AI)

Technology that enables computers to perform tasks that normally require human intelligence, such as recognizing images, understanding language, or making predictions.

---

## API

An Application Programming Interface that allows two software applications to communicate with each other.

---

## B

## Bug

An error or defect in software that causes unexpected behavior.

---

## Boolean

A data type that has only two possible values:

- True
- False

---

## C

## Class

A blueprint used to create objects in Object-Oriented Programming.

---

## Constructor

A special method used to initialize an object when it is created.

---

## CSS

Cascading Style Sheets used to control the appearance and layout of web pages.

---

## Cybersecurity

The practice of protecting computers, networks, and data from unauthorized access or attacks.

---

## D

## Data

Information that can be stored, processed, and analyzed by computers.

---

## Debugging

The process of identifying and correcting errors in software.

---

## Documentation

Written information that explains how software or a project works.

---

## DOM

The Document Object Model, which represents the structure of an HTML document and allows JavaScript to modify webpage content.

---

## E

## Encapsulation

An Object-Oriented Programming principle that groups data and methods together while controlling access.

---

## F

## File Handling

Reading from and writing data to files using a programming language.

---

## Function

A reusable block of code designed to perform a specific task.

---

## G

## Git

A distributed version control system used to track changes in source code.

---

## GitHub

An online platform used to host Git repositories and collaborate on software projects.

---

## H

## HTML

HyperText Markup Language used to structure web pages.

---

## I

## Internet of Things (IoT)

A network of physical devices connected to the internet that can collect and exchange data.

---

## Inheritance

An Object-Oriented Programming concept where one class can inherit properties and behaviors from another.

---

## J

## JavaScript

A programming language used to create interactive web pages.

---

## L

## Library

A collection of reusable code that provides additional functionality.

---

## M

## Machine Learning

A branch of Artificial Intelligence where computers learn patterns from data to make predictions or decisions.

---

## Method

A function that belongs to a class.

---

## Module

A Python file containing reusable code that can be imported into other programs.

---

## O

## Object

An instance created from a class.

---

## Object-Oriented Programming (OOP)

A programming paradigm based on classes and objects.

---

## P

## Portfolio

A collection of a student's projects, documentation, reflections, and achievements demonstrating learning progress.

---

## Prompt Engineering

The practice of designing effective instructions for Artificial Intelligence systems to produce useful responses.

---

## Prototype

An early working version of a product or system used for testing and improvement.

---

## Python

A high-level programming language known for its simplicity and versatility.

---

## R

## Repository

A storage location containing project files and version history managed by Git.

---

## Responsive Design

A web design approach that allows websites to adapt to different screen sizes.

---

## Robotics

The field of designing, building, programming, and operating robots.

---

## S

## Sensor

An electronic component that detects changes in the environment, such as light, distance, or temperature.

---

## Software Engineering

The systematic process of designing, developing, testing, and maintaining software.

---

## Source Code

Human-readable programming instructions written by developers.

---

## Sprint

A short, planned period of work used to complete specific project goals.

---

## T

## Testing

The process of verifying that software or hardware works as expected.

---

## Troubleshooting

The systematic process of identifying and resolving technical problems.

---

## U

## User Interface (UI)

The visual components through which users interact with a software application.

---

## V

## Version Control

The practice of tracking and managing changes to software files over time.

---

## W

## Web Development

The process of building and maintaining websites and web applications.

---

## Workflow

A structured sequence of tasks performed to complete a project or process.
