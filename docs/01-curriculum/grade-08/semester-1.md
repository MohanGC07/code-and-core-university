## Grade 8 Semester 1 Learning Plan

## Purpose

Semester 1 strengthens students' software development skills. Learners transition from writing individual programs to designing structured software applications using professional development practices.

---

## Semester Theme

## Professional Software Development

Students learn to:

- Design software before coding.
- Build modular Python applications.
- Create responsive websites.
- Apply software engineering principles.
- Work collaboratively on technical projects.

---

## Semester Structure

| Weeks | Unit | Focus |
| ------- | ------ | ------- |
| 1–4 | Unit 1 | Advanced Python Programming |
| 5–8 | Unit 2 | Object-Oriented Programming |
| 9–12 | Unit 3 | Responsive Web Development |
| 13–18 | Unit 4 | Software Engineering Fundamentals |

---

## Unit 1 — Advanced Python Programming

## Learning Objectives

Students will:

- Organize Python projects.
- Create reusable modules.
- Handle exceptions.
- Read and write files.
- Improve code readability.

### Topics

- Modules
- Packages
- Exception Handling
- File Handling
- Code Organization
- Python Best Practices

### Practical Activities

- Contact Manager
- Student Record System
- Expense Tracker
- File Organizer

### Deliverables

- Python Programs
- Documentation
- Reflection Notes

---

## Unit 2 — Object-Oriented Programming

## Learning Objectives — Object-Oriented Programming

Students will:

- Understand classes and objects.
- Create reusable software components.
- Design simple software systems.
- Apply encapsulation principles.

### Topics — Object-Oriented Programming

- Classes
- Objects
- Constructors
- Methods
- Attributes
- Encapsulation

### Practical Activities — Object-Oriented Programming

- Library Management System
- Banking Simulation
- Student Management System

### Deliverables — Object-Oriented Programming

- OOP Applications
- UML Sketches
- Technical Documentation

---

## Unit 3 — Responsive Web Development

## Learning Objectives — Responsive Web Development

Students will:

- Build responsive websites.
- Apply Flexbox and CSS Grid.
- Create interactive webpages.
- Improve website accessibility.

### Topics — Responsive Web Development

- Responsive Design
- Media Queries
- Flexbox
- CSS Grid
- Forms
- JavaScript Basics
- DOM Introduction

### Practical Activities — Responsive Web Development

- Responsive Portfolio
- School Website
- Interactive Quiz
- Product Landing Page

### Deliverables — Responsive Web Development

- Responsive Websites
- JavaScript Exercises
- Design Documentation

---

## Unit 4 — Software Engineering Fundamentals

## Learning Objectives — Software Engineering Fundamentals

Students will:

- Understand the software development lifecycle.
- Organize software projects.
- Apply documentation standards.
- Understand version control concepts.
- Perform basic software testing.

### Topics — Software Engineering Fundamentals

- SDLC
- Requirements
- Planning
- Documentation
- Git Concepts
- Testing Fundamentals

### Practical Activities — Software Engineering Fundamentals

- Project Planning
- Documentation Exercise
- Testing Checklist
- Team Planning Activity

### Deliverables — Software Engineering Fundamentals

- Software Plan
- Technical Documentation
- Testing Report

## Semester Capstone Project

## Team Productivity Application

Students collaborate to design and build a software application that improves productivity in school or daily life.

### Example Projects

- Homework Planner
- School Library Manager
- Classroom Attendance System
- Event Management System
- Student Task Tracker

### Required Features

- Well-organized code
- File storage
- User-friendly interface
- Error handling
- Documentation
- Final presentation

---

## Teaching Methodology

Teachers should emphasize:

- Project-based learning
- Pair programming
- Guided practice
- Collaborative problem-solving
- Design before implementation
- Continuous feedback

## Portfolio Requirements

Students should include:

- Python Applications
- OOP Projects
- Responsive Websites
- JavaScript Activities
- Software Documentation

## Expected Outcomes

By the end of Semester 1, students should be able to:

- Build structured Python applications.
- Apply object-oriented programming principles.
- Develop responsive websites.
- Understand software engineering fundamentals.
- Collaborate effectively on software projects.
- Produce professional technical documentation.
