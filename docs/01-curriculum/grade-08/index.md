## Grade 8 Curriculum Overview

## Purpose

Grade 8 marks the transition from learning technology to engineering technology. Students strengthen their programming skills, begin building larger software systems, explore Artificial Intelligence in greater depth, develop responsive websites, and build intelligent robotics solutions.

The curriculum emphasizes analytical thinking, creativity, collaboration, ethical technology use, and professional software development practices.

---

## Grade Theme

## Building Intelligent Systems

Students begin thinking like software developers and junior engineers by designing, developing, testing, documenting, and presenting technology solutions.

---

## Academic Vision

By the end of Grade 8, every learner should be able to:

- Design structured software solutions.
- Develop responsive websites.
- Build intelligent robotics systems.
- Explain the AI development lifecycle.
- Apply engineering design principles.
- Work collaboratively on technical projects.
- Maintain professional documentation.
- Present technical work confidently.

---

## Curriculum Philosophy

The Grade 8 curriculum follows these principles:

- Learn by Building
- Think Like an Engineer
- Solve Real Problems
- Collaborate Professionally
- Design Before Coding
- Learn Through Reflection
- Build Ethical Technology
- Continuously Improve

## Learning Progression

Grade 8 builds directly upon Grade 7.

| Grade 7 | Grade 8 |
| ---------- | ---------- |
| Intermediate Python | Object-Oriented Python |
| HTML & CSS | Responsive Web Development |
| AI Concepts | AI Development Workflow |
| Arduino Basics | Intelligent Automation |
| Basic Documentation | Professional Documentation |
| Individual Projects | Team Engineering Projects |

## Learning Methodology

Every lesson follows the Code and Core Engineering Cycle.

1. Learn
2. Analyze
3. Design
4. Build
5. Test
6. Improve
7. Document
8. Present
9. Reflect

## Graduate Profile

A student completing Grade 8 should demonstrate:

- Confidence in solving technical problems.
- Ability to build complete software projects.
- Understanding of AI fundamentals.
- Competency in responsive web development.
- Ability to design robotics solutions.
- Professional documentation habits.
- Effective collaboration and communication.
- Ethical and responsible technology use.
