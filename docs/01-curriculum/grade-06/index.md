## Grade 6 Curriculum Overview

## Theme

### Create with Technology

Grade 6 builds on the Grade 5 foundation by introducing structured programming, beginner electronics, Arduino, computational thinking, and responsible AI. Students begin transitioning from block-based programming to text-based programming using Python.

---

## Grade Profile

| Attribute | Description |
| ----------- | ----------- |
| Grade | 6 |
| Academic Stage | Foundation |
| Typical Age | 11–12 Years |
| Programming Level | Beginner → Early Intermediate |
| Learning Style | Project-Based |

---

## Annual Goal

By the end of Grade 6 learners will be able to:

- Build simple Python programs.
- Explain how algorithms solve problems.
- Create Arduino-based electronic projects.
- Understand AI through practical examples.
- Work in teams on technology projects.
- Document and present their work.
- Build confidence for Grade 7.

---

## Learning Outcomes

## Digital Literacy

Students can:

- Manage files and folders.
- Use cloud storage.
- Practice online safety.
- Understand digital citizenship.

---

## Computational Thinking

Students can:

- Write algorithms.
- Create flowcharts.
- Break problems into steps.
- Debug simple programs.

---

## Programming

Students learn:

- Python Introduction
- Variables
- Input / Output
- Data Types
- Arithmetic Operators
- Conditions
- Loops
- Functions (Introduction)

Platforms:

- Python
- Scratch (Advanced)
- PictoBlox

---

## Artificial Intelligence

Students explore:

- AI vs Human Intelligence
- Data and AI
- AI Ethics
- Image Recognition
- Voice Recognition

---

## Robotics & Electronics

Students learn:

- Arduino Basics
- LEDs
- Buttons
- Buzzers
- LDR
- Ultrasonic Sensor (Introduction)
- Servo Motor (Introduction)

## Creativity & Design

Students practice:

- Storytelling
- Digital Posters
- UI Sketching
- Design Thinking

---

## Communication & Collaboration

Students learn:

- Teamwork
- Public Speaking
- Project Demonstration
- Reflection Writing

## Portfolio Requirements

Every learner maintains:

- Minimum 8 Python Programs
- 3 Arduino Projects
- Reflection Journal
- Project Documentation
- Photos and Videos
- Presentation Slides
