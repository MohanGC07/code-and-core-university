# GRADE-006-002 — Learning Outcomes

## Domains

### Digital Literacy

Students can:

- Organize files and folders  
- Use cloud storage safely  
- Practice password security  
- Explain digital citizenship  

### Computational Thinking

Students can:

- Write algorithms  
- Design flowcharts  
- Break problems into steps  
- Debug simple programs  

### Programming

Students can:

- Write Python programs  
- Use variables  
- Take user input  
- Perform calculations  
- Use conditions  
- Use loops  
- Create reusable functions  
- Debug programs  

### Artificial Intelligence

Students can:

- Explain AI using examples  
- Describe how AI learns from data  
- Discuss responsible AI use  
- Recognize AI in everyday life  

### Robotics

Students can:

- Build Arduino circuits  
- Upload Arduino programs  
- Read sensor values  
- Control LEDs and buzzers  
- Build simple automation systems  

### Communication

Students can:

- Present projects  
- Work in teams  
- Document their learning  
- Reflect on successes and challenges
