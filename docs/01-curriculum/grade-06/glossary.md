# GRADE-006-014 — Glossary

| Term | Definition |
| ------ | ---------- |
| Algorithm | A sequence of steps to solve a problem. |
| Variable | A named place to store data. |
| Loop | A programming structure that repeats instructions. |
| Function | A reusable block of code. |
| Sensor | A device that detects changes in its environment. |
| Arduino | A microcontroller platform for electronics and programming. |
| Artificial Intelligence | Technology that enables computers to perform tasks requiring human-like intelligence. |
| Debugging | Finding and fixing errors in a program. |
