## Grade 5–12 Curriculum Matrix

## Purpose

The Curriculum Matrix provides a clear overview of what learners study each academic year.

It ensures every grade builds naturally upon previous knowledge while preparing learners for advanced technologies and future careers.

---

## Learning Progression

| Stage | Grades | Focus |
| -------- | -------- | ------- |
| Foundation | Grade 5–6 | Curiosity, digital literacy, logical thinking |
| Explorer | Grade 7–8 | Programming, robotics, web technologies |
| Builder | Grade 9–10 | Software engineering, AI foundations |
| Professional | Grade 11–12 | AI, full-stack engineering, research, leadership |

---

## Learning Philosophy

Every learner will:

- Learn concepts
- Practice regularly
- Build projects
- Reflect on progress
- Maintain a portfolio
- Collaborate with peers
- Present completed work

---

## Expected Graduate Profile

By the end of Grade 12, graduates should be able to:

- Develop full-stack applications.
- Build AI-powered solutions.
- Design robotics systems.
- Use Git and collaborative workflows.
- Deploy modern applications.
- Maintain a professional portfolio.
- Continue confidently into higher education or technology careers.

---

## Continuous Review

The Curriculum Matrix is reviewed annually to reflect:

- Industry trends
- Educational research
- Technology evolution
- Instructor feedback
- Student outcomes
