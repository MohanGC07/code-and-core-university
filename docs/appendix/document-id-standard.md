# Document ID Standard

Every official document must have a unique identifier.

| Prefix | Meaning |
| --------- | --------- |
| COM | Company |
| CUR | Curriculum |
| SCH | School |
| PRG | Program |
| CRS | Course |
| MOD | Module |
| LES | Lesson |
| PRJ | Project |
| ASM | Assessment |
| YTB | YouTube |
| WEB | Website |
| BUS | Business |
| OPS | Operations |
| SOP | Standard Operating Procedure |
| TMP | Template |

Examples:

- COM-001
- CUR-001
- CRS-015
- LES-120
- PRJ-042
